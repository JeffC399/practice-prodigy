# Practice Prodigy

A pro-quality, modern practice platform for musicians. Launching with a focused **Bass Arpeggios** module — drill arpeggio patterns over user-defined chord sequences against a precision metronome, with controllable look-ahead, count-in, and tempo. Built to expand into a comprehensive multi-instrument practice ecosystem.

> Status: **pre-build** (v0.1 — design complete; scaffolding next).

## v1 in One Sentence

A polished web + PWA app for bass players to drill Scale Tones / Arpeggiated 7ths / 1-3-5-LT / 8-7-5-3 patterns over any sequence of chords (15+ jazz qualities, 8 ordering strategies including Cycle of 5ths and several randomization modes), with controllable rhythm, look-ahead, and count-in.

## Tech Stack (Summary)

Next.js 16 (App Router) · TypeScript · Tailwind CSS 4 · shadcn/ui · Framer Motion · Zustand · Dexie (IndexedDB) · Tone.js · tonal.js · Lucide React · Geist typography. Deployed on Vercel from `main`. Capacitor (mobile) and Tauri (desktop) wrappers post-v1. Supabase (Auth + Postgres + Storage) integrated in v1.1+.
